Performance Module Help

Overview

The Performance module provides real-time monitoring and optimization capabilities for your Jitsi Meet Qt application.

Features

  • Real-time CPU, Memory, and Network monitoring
  • Performance optimization recommendations
  • Customizable alerts and thresholds
  • Export performance data to CSV/HTML reports

Getting Started

To start monitoring performance:

  1. Open the Performance panel
  2. Click "Start Monitoring"
  3. Configure your monitoring preferences
  4. View real-time metrics in the dashboard
